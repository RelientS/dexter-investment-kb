# AI 应用爆发与商业化落地

> Agent、SaaS 订阅、端侧 AI、手机 / PC 换机、机器人具身、自动驾驶 FSD 等下游应用的变现节奏、客单价逻辑与杀手级应用的识别框架。

## 核心论点

- AI 算力需求天梯从近到远是云 GPU、云 DRAM、云 NAND、云 CPU、机器人 IoT、本地软件 Agent,每一层都是可持续增量
- 机器人和 IoT 是 AI 算力天梯倒数第二层,基本面跟上估值的时点靠后
- 本地软件 Agent 是 AI 算力天梯最远的终端层
- Web4 就是 Agentic Web,agent 将取代手机 app launcher 成为下一代互联网入口
- 在通用 agent 里集成软件才是下一代范式,在软件里集成 agent 意味着收缩、关停或吞并
- 腾讯萎靡的原因是想把 Agent 集成到已有软件而非反过来,方向错误
- 订阅模式本质是 SaaS 时代的捆绑销售和补贴模式,正在被取代
- GPU服务器软件层清零大家同一起跑线,公有云不掌握agent和全栈软件层会被neocloud冲击
- Cerebras相当于奥特曼个人投资的neocloud,新模型专门为其硬件设计
- 腾讯是AI时代最危险的巨头之一,如能筛选会从恒科中剔掉腾讯美团百度快手金山
- 买Amazon应该为了机器人而非Trainium,闪迪是正经成长股不是meme股,POET是妖股
- Cerebras靠前道工艺做超大芯片,华为靠芯片间通信+软件软硬架构创新
- 只要价值来源是减少摩擦,不管是软件/SaaS还是真人岗位都会因AI被消灭
- Agent开发被前端夺舍了
- 只有劳动力模式Agentic AI(Anthropic)不能让大众更美好,消费者AI才能增强大众
- 智谱估值应按ARR和未来预期收入算,增速和毛利率在快速改善
- 京东做AI应该主要做Tool而非agent,agent是实现新tool的手段
- Anthropic营收数据有两大水分,接下来会大波动可能牵连美股AI大跌
- 腾讯小程序Agent方案类似opencli操作web,缺乏明确的Tool定义,只能从HTML/UI结构视觉操作并提取服务端API,实现效果难且token消耗大
- 可将宇树人形机器人的技术架构与人脑结构做映射类比:操作者/编舞对应大脑皮层前额叶,RL策略对应基底节运动皮层,全身协调空翻对应小脑,平衡步态对应脑干,关节伺服对应脊髓,传感器对应感觉系统
- 叠衣服类人形机器人操作比翻跟头类技术难度更高,因为需要更多大脑皮层级别的认知和精细控制能力
- 3D点云(3DSG)是当前最热门的机器人感知技术方向
- C端AI需要少数「把软件嵌入agent里」的通用agent企业,通过新分发入口、广告、电商等新商业模式来支撑全球大部分C端token消耗
- B端AI的出路要么是企业用token替代人力实现裁员,要么由少数高效的AI企业软件公司集中消耗B端大部分token
- 汽车本质就是机器人,当前作为全球经济支柱的汽车产业最终将平移为机器人产业,上下游和关键技术也将随之替换
- 当前AI行业正处于5%到30%渗透率的跨越阶段,而机器人行业还未达到5%的early adopter阶段
- 黄仁勋判断机器人技术将是韩国下一个支柱产业
- OpenAI近期代码提交量大幅增加,但其CICD环境遇到瓶颈,正试图摆脱对GitHub的依赖;GitHub的设计和基础设施开始难以适应agent主导的软件开发生态
- 前端的本质定义是「直接接触用户的最高层级软件抽象层」,当用户从人类变成agent后,前端就变成了做agent相关的工作;前端的根本不是用户,而是最高层级的抽象层,即使没有真人用户它也必然存在
- 很多垂直agent和集成在现有软件里的agent是伪需求,只有通用agent的需求是刚性且注定少不了的
- Agent用的Tool分服务器端(MCP、Skill封装的Open API)和客户端(CLI、WebMCP/小程序MCP、App Intents等),harness框架也对应分两端
- 全套客户端agent能力做得好就是Agentic OS,做得差就是山寨ChatGPT加SaaS式AI能力集成没有本质差异
- 云端AI必须跟端侧AI配合,类似人形机器人架构;微信Agentic小程序就是端侧多模态小模型充当云端agent与客户端Tool的翻译
- AI的根本问题不是消费能力,而是能否像iPhone那样无中生有创造新需求和新现金流
- 除AI外的出口不可持续,新能源、机器人、全产业链都是AI基建驱动的;没了AI中国出口只剩南方国家市场
- 串流视频和AI agent虽是软件,但不具备边际成本几乎为零的软件核心优势
- 按量付费的AI token消费在公司环境(如字节)里很爽,但离职后固定订阅/token消费会变成压在身上的负债,此时AI能力不再被视为个人能力的一部分
- GenUI-first Web(设备从「安装软件」走向「生成软件」)规模庞大有乘数效应,但端到端「日抛软件」若不建在新抽象架构上没价值,本质是tokenmaxxing
- Agent按量付费类比人类社会就是律师按小时付费的高端服务业模式,不能类比搬砖/计件
