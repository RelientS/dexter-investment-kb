# AI 上游算力与半导体产业链

> GPU / HBM / 光模块 / 液冷 / 交换机硬件主线；英伟达 CUDA + Blackwell 生态、台积电 CoWoS 先进封装、Fabless 与代工制造的产业链分工，以及涨价与产能周期。

## 核心论点

- SanDisk 是 2024 年股王涨 +557%,元旦后再涨 +145%,SSD/NAND 是产品不是商品,可绕开 HBM 瓶颈
- 长鑫存储预计是今年 A 股股王,HBM2 起步值得打新
- 美股四大支柱是 NVDA、TSLA、Cloudflare、SanDisk,应长期持有
- AI 算力需求天梯从近到远是云 GPU、云 DRAM、云 NAND、云 CPU、机器人 IoT、本地软件 Agent,每一层都是可持续增量
- 云 GPU 是 AI 算力需求核心红利,包括 NVDA 加 NeoCloud
- 云 DRAM 和 HBM 估值已含 2027 年增长,下游变数大,已进入鱼尾阶段
- 云 NAND 即 SSD 是产品而非商品,可绕开 HBM 瓶颈,代表是 SanDisk 和闪迪
- 云 CPU 需求会因 agent swarm 加 Tool Use 指数级增长,AMD 正在建仓
- OpenAI 的路线是 Codex app 发展成桌面唯一超级 app,吞并 Atlas 加 ChatGPT
- 字节豆包有风险,纯问答模式无资产沉淀,用户替换成本为 0
- 桌面 agent 会让中国完成之前中断的 PC 普及,PC 将迎来复兴
- API 补贴模式危险,用多亏多,无法长期持续
- 按量付费模式可持续性存疑,企业会像限制 headcount 一样限制 agent 用量
- Anthropic 可能短视取巧,OpenAI 则押注 100% 身家在下一代浏览器加对抗 Google
- H200 是中国 AI 唯一瓶颈,国产连 HBM3 都还在攻关
- HBM 没有差异化,只有能做和不能做的区别,技术壁垒非技术领先优势
- DRAM 是 AI 公敌,阻碍 scaling law,会被下游严厉惩罚
- 推理 PD 分离将成为趋势,prefill 用 GDDR7,decode 用 SRAM 替代 HBM
- coding 是 agent 必经之路,跟是否达到 AGI 没关系
- AGI 必须是圆形(全能力均衡),不能局部脑残
- 统一通用 Runtime 加生态是方向,Bun 这类 App Runtime 能减少 context 切换
- 裁员是 AI 公司的利好信号,DeepL25%、CF20%、Upwork25%、Atlassian10% 裁员后财报暴涨
- GPU 是最难卷的环节,国产算力先指望推理侧突破
- Qwen(千问)不能偏科,而 GLM 和 MiniMax 可以走 Claude 的偏科路线
- 智谱的护城河在体制内市场,公务员和国企员工也需要企业 agent
- MCP 本质是 Web 生态,国内目前空白,千问直接接入阿里全家桶(高德闪购支付宝)很危险
- 英伟达边缘算力产品一直有在销售
- 边缘算力受带宽限制,适合跑本地多模态
- SemiAnalysis是专业度很高的半导体分析机构,创始人29岁员工85人
- Cerebras、LPU/Groq等是业界摆脱HBM瓶颈的努力,高速推理越大HBM可持续性越低
- 摆脱HBM的两条技术路径:LPU/Groq靠软件编译,Cerebras靠硬件大芯片
- 转岗方向应选择教人用AI
- GPU服务器软件层清零大家同一起跑线,公有云不掌握agent和全栈软件层会被neocloud冲击
- GPU和光通信是AI scaling law的一部分,HBM只与特定架构有关卡脖子状态不可持续
- 光通信在AI基建中用量持续增长,水平扩展和垂直扩展都要增加光的使用
- 硅光、光IO、异构计算、SRAM、TPU、英伟达推理方案都在减少对HBM的依赖
- 不是所有AI需求都需要长上下文,Cerebras侧重速度优先的场景
- 英伟达和HBM估值低是因为市场担忧营收利润率增速不可持续,HBM本质仍是周期股
- 台积电是价值股,光模块厂要变成长股必须发展硅光等新技术,三家光模块厂策略不同
- OpenAI不亏更多是减少再投资可分红,成长股投资者会撤价值投资者会进场
- 光通信新技术更适合半导体企业做,传统光模块厂面临的是赛道切换而非门槛竞争
- 字节传统业务和新业务都受益于AI,做公有云晚在CPU/GPU比例上反而有减计优势
- Oracle做neocloud的核心优势是沉淀了大量传统行业数据
- Anthropic按量付费短期有大量浪费,企业会像限制headcount一样限制agent用量
- 互联网泡沫和现在AI基建的核心区别是供给侧驱动vs消费侧驱动(算力永远不够)
- 字节广告利润率达Meta/Google级别,AI资本开支更容易赚回来,与腾讯不同
- 智谱和MiniMax虽然AI很纯但有其他风险,并非真的中国版Anthropic和OpenAI
- 推算OpenAI明年初上市市值会到一万亿以上,1万亿太保守
- 英伟达核心是加速更底层通用计算,用CUDA生态找重复性再硬件并行

- CUDA的真正优势是能轻易获得性能优化,ASIC每次优化都要重新建产线

- Cloudflare是agent infra和Tool infra,不是算力infra,NET已持有多年

- Cerebras靠前道工艺做超大芯片,华为靠芯片间通信+软件软硬架构创新

- 字节跑通了消费者AI变现闭环,漫剧比真人短剧更能发挥GenAI优势
- 豆包最大难点是创造用户对AI的持续需求,套壳和产品包装很重要
- 英伟达CPX(同构大context)和LPX(GPU+LPU异构速度)是两条不同的Rubin路径

- 豆包幻觉多的根因是产品设计默认使用快速模式，并非模型能力本身的问题

- NVL576带来的新增光需求无法用传统可插拔光模块满足，对缺少硅光能力的厂商是实质性利空
- ASIC比GPU更依赖跨机柜大规模互联，因此对光互联的需求发展更快
